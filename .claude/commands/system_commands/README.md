# ⚙️ System Commands

System management and optimization commands

## Available Commands

- `/system:health-check` - Health-Check command
- `/system:optimize` - Optimize command
- `/system:migrate` - Migrate command
- `/system:prime-core` - Prime-Core command

## Usage

All system commands follow the pattern:
```
/system:[command] [OPTIONS]
```

## Examples

```bash
/system:health-check --help
```

```bash
/system:optimize --help
```

```bash
/system:migrate --help
```
